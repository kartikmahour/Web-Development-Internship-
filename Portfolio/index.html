  • Project 2

    Gym Registration Form Webpage

    Description:

    • This is a simple webpage designed to allow users to register for a gym membership. The webpage features a clean and user-friendly layout with a registration form that asks users to provide their name, age, gender, locality, email address, and phone number.
    • The HTML code for the webpage is well-structured and organized, using semantic HTML tags such as header, nav, form, and input to provide proper context to the content. The form includes a JavaScript function to validate the phone number input and ensure that it is 10 digits long.
    • The webpage design is responsive, adapting to different screen sizes and devices. The CSS styling is separated from the HTML code in a separate file, which promotes clean and maintainable code.
